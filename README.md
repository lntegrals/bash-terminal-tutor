# Bash Terminal Tutor 🖥️

An interactive web app to learn bash terminal commands built with Marimo.

## Features

- **Interactive Lessons**: Learn essential bash commands step by step
- **Live Terminal**: Practice commands in a simulated terminal environment
- **Progress Tracking**: Track your learning progress
- **Exercises**: Test your knowledge with hands-on exercises

## Quick Start

### Run Locally

```bash
pip install marimo
marimo tutor bash_tutor.py
```

### Deploy to GitHub Pages

1. Fork this repository
2. Go to Settings → Pages
3. Set Source to "Deploy from a branch"
4. Select `gh-pages` branch
5. Visit your deployed site at `https://yourusername.github.io/bash-terminal-tutor/`

## Lessons

1. **Navigation**: `ls`, `cd`, `pwd`
2. **Files**: `mkdir`, `touch`, `rm`, `cp`, `mv`
3. **Reading**: `cat`, `head`, `tail`, `less`
4. **Search**: `grep`, `find`
5. **Permissions**: `chmod`, `chown`
6. **Pipes & Redirection**: `|`, `>`, `>>`, `<`
